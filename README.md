# 🛒 E-Commerce Product Recommendation Engine

## 📌 Project Overview
This project is a **DSA-based + ML-inspired E-Commerce Product Recommendation System** built using Python and Streamlit.

It recommends products to users based on:
- Purchase history
- Search history
- Cart behavior
- Product similarity
- Collaborative filtering (co-occurrence)

---

## 🎯 Problem Statement
E-commerce platforms like Amazon and Flipkart need intelligent systems to recommend products to users to increase engagement and sales.

This project simulates that system using Data Structures and Algorithms.

---

## ⚙️ Tech Stack
- Python
- Streamlit (UI)
- heapq (Priority Queue)
- Dictionaries (HashMap)
- JSON datasets

---

## 🧠 Concepts Used (DSA)
- HashMap (Dictionary storage)
- Heap (Priority Queue)
- Greedy Ranking
- Graph-like co-occurrence matrix
- Sorting & scoring system

---

## 🤖 ML Concept Used
- Content-based filtering
- Collaborative filtering (co-occurrence matrix)
- Hybrid scoring model

---

## 🏗️ System Architecture

User Data + Product Data  
→ HashMap Storage  
→ Feature Extraction  
→ Hybrid Scoring  
→ Heap (Top-K selection)  
→ Recommendation Output
